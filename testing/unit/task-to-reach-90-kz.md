# Cart — Техникалық тапсырма

## Жоба: Food To Go

### Мақсат

`Cart` класын жасау, ол пайдаланушының себетін басқаруға арналған: тауарларды қосу/алу, промокодтарды қолдану және жинақтық жеңілдіктермен соңғы соманы есептеу.

Класс **таза** болуы керек, сыртқы тәуелділіктерсіз, толықтай юнит-тесттермен тексерілуі мүмкін.

---

## 1. Класс негізгі функциялары

### 1.1. Тауар қосу

Метод: `void addItem(String itemName, int quantity)`

* Көрсетілген санды себетке қосады.
* Егер тауар менюде жоқ болса → `IllegalArgumentException`.

### 1.2. Тауар алып тастау

Метод: `void removeItem(String itemName, int quantity)`

* Себеттегі тауар санын азайтады.
* Егер тауар жоқ немесе сан көп болса → `IllegalArgumentException`.

### 1.3. Промокод қолдану

Метод: `void applyPromoCode(String promoCode)`

* Рұқсат етілген промокодтар:

  * `FOOD10` → егер сумма ≥ 2000₸ болса, 10% жеңілдік
  * `FOOD20` → егер сумма ≥ 5000₸ болса, 20% жеңілдік
* Жарамсыз промокод → `IllegalArgumentException`.

### 1.4. Соңғы соманы есептеу

Метод: `double calculateTotal()`

* Сомасы = барлық тауарлардың санына байланысты есептеледі.
* Қолданылған промокод есепке алынады.
* OrderService жеңілдіктері қолданылады: ≥ 5000₸ → 10%, ≥ 10000₸ → 15%.

### 1.5. Қысқаша есеп алу

Метод: `String getSummary()`

* Шығару мыналарды қамтиды:

  * Тауар саны
  * Жеңілдікке дейінгі сумма
  * Қолданылған жеңілдік/промокод
  * Соңғы сома

Мысалы:

```
Cart summary: 5 items, subtotal = 5400₸, discount = 10%, total = 4860₸
```

---

## 2. Тестілеу талаптары

* **Чистые юнит-тесттер**, моктарсыз.
* Барлық әдістер мен жағдайларды тексеру:

  * Тауарларды қосу және алу
  * Рұқсат етілген/рұқсат етілмеген промокодтарды қолдану
  * Жеңілдіктер мен промокодтарды есепке алу
  * Қысқаша есептің дұрыс қалыптасуын тексеру

Тест сценарийлері:

* Тауар қосу (0 немесе теріс сан қосу жағдайларын қоса)
* Себеттегі саннан көп тауарды алу → қателік
* Минималды сомадан төмен промокод қолдану → қателік
* Бірнеше промокод қолдану → соңғы ғана есепке алынады

## 3. Бағалау критерийлері

| Критерий                                       | Балл |
| ---------------------------------------------- | ---- |
| Тауарларды қосу/алу логикасын жүзеге асыру     | 3    |
| Промокодтар мен жинақтық жеңілдіктерді қолдану | 3    |
| Соңғы соманың дұрыс есептелуі                  | 2    |
| Қысқаша есептің дұрыс қалыптасуы               | 2    |
| Тесттермен толық қамтылуы                      | 2    |
| Код тазалығы және әдіс атаулары                | 1    |
| **Барлығы**                                    | 10   |
